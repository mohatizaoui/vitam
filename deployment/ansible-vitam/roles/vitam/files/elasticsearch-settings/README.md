Attention
=========

Le fichier elasticsearch-configuration.json est le fichier de configuration d'elasticSearch assurez-vous si vous l'avez modifié d'appliquer la procédure de mise à jour avec le tag ```update_vitam_configuration```

Ce fichier est déployé dans les services suivants :

- Metadata
- Metadata-Collect
- Functional-administration
- logbook
- ihm-recette
